BAGIAN 1: HARD SKILLS (KLASIK)

"Jangan pernah meremehkan skill ini. Ini adalah 'Bread and Butter' seorang IE. Tanpa ini, Anda hanya Data Analyst, bukan Engineer."

01. Work Design & Measurement

  • Time Study: Pengukuran waktu kerja menggunakan stopwatch (metode langsung).
  • Motion Study: Analisis gerakan mikro (Therblig) untuk menghilangkan gerakan tidak perlu.
  • Work Sampling: Mengukur produktivitas tanpa mengamati terus-menerus.
  • Standard Time Calculation: Menghitung waktu baku dengan faktor penyesuaian (rating factor) dan kelonggaran (allowance).
  • MOST / MTM: Predetermine time systems (menghitung waktu kerja berdasarkan tabel standar gerakan, tanpa stopwatch).

02. Quality Control & Management

  • 7 QC Tools: Check sheet, Pareto chart, Fishbone diagram, Histogram, Scatter diagram, Stratification, Control chart.
  • Statistical Process Control (SPC): Menghitung Cp, Cpk, dan memantau variabilitas proses.
  • Six Sigma (DMAIC): Metodologi Define, Measure, Analyze, Improve, Control.
  • Total Quality Management (TQM): Filosofi kualitas menyeluruh.
  • ISO Standards: Pemahaman audit ISO 9001 (Mutu), ISO 14001 (Lingkungan), ISO 45001 (K3).
  • Acceptance Sampling: Menentukan jumlah sampel lot untuk inspeksi masuk/keluar.

03. PPIC

  • Forecasting (Klasik): Moving Average, Exponential Smoothing, Linear Regression.
  • Inventory Management: EOQ (Economic Order Quantity), ROP (Reorder Point), Safety Stock calculation.
  • MRP (Material Requirements Planning): Bill of Materials (BOM), Master Production Schedule (MPS).
  • Capacity Planning: Rough Cut Capacity Planning (RCCP), Capacity Requirements Planning (CRP).
  • Scheduling: Job shop scheduling, Flow shop scheduling, Gantt Charting, Line Balancing.

04. Supply Chain & Logistics

  • Warehouse Layout: Perancangan tata letak gudang, manajemen lokasi (Bin management).
  • Transportation Problem: VRP (Vehicle Routing Problem), Milk-run system.
  • Purchasing & Procurement: Negosiasi vendor, pemilihan supplier.
  • Incoterms: Memahami istilah perdagangan internasional (FOB, CIF, EXW).

05. Ergonomi & Human Factors

  • Anthropometry: Pengukuran dimensi tubuh untuk desain alat kerja.
  • Biomechanics: Analisis beban angkat (NIOSH Lifting Equation), analisis postur (RULA, REBA).
  • Physiology of Work: Pengukuran beban kerja fisik (denyut jantung, konsumsi oksigen).
  • Environmental Ergonomics: Pencahayaan, kebisingan, temperatur, dan getaran.

06. Ekonomi Teknik

  • Cost Accounting: HPP (Harga Pokok Produksi), Activity Based Costing (ABC).
  • Investment Analysis: NPV (Net Present Value), IRR (Internal Rate of Return), Payback Period, ROI.
  • Depreciation: Perhitungan penyusutan aset (Garis lurus, saldo menurun).
  • Break-Even Analysis: Menghitung titik impas.

07. Operations Research (Klasik/Deterministik)

  • Linear Programming: Simplex method, sensivitas analisis.
  • Integer Programming & Goal Programming.
  • Transportation & Assignment Models.
  • Network Planning: CPM (Critical Path Method), PERT.

BAGIAN 2: MODERN STEM & INDUSTRY 4.0

"Ini adalah skill yang membuat Anda dibayar mahal di era digital dan membedakan Anda dari lulusan manajemen."

01. Data Science & Advanced Analytics

  • Programming Languages: Python (Wajib: Pandas, NumPy), R.
  • Database Management: SQL (Querying, Join, Aggregate), NoSQL (MongoDB - dasar).
  • Data Visualization: Tableau, Power BI, Google Looker Studio (Dashboarding interaktif).
  • Machine Learning (Supervised): Regression, Classification (Decision Trees, Random Forest).
  • Machine Learning (Unsupervised): Clustering (K-Means untuk segmentasi customer), Association Rules.
  • Big Data Tools: Dasar-dasar Hadoop atau Spark (untuk data skala masif).

02. Advanced Simulation & Digital Twin

  • Discrete Event Simulation (DES): Arena, FlexSim, AnyLogic, ProModel (Logika simulasi stokastik).
  • Agent-Based Modeling: Memodelkan perilaku individu otonom dalam sistem.
  • System Dynamics: Vensim, Stella (Memodelkan loop umpan balik/feedback loops).
  • Digital Twin Creation: Menghubungkan model simulasi dengan data real-time sensor.

03. Industrial Automation & IoT (IIoT)

  • Sensor Knowledge: Memahami cara kerja sensor (proximity, vision, temperature).
  • PLC Basics: Memahami Ladder Logic atau Functional Block Diagram.
  • Connectivity Protocols: MQTT, Modbus, OPC-UA (Cara mesin "berbicara" ke server).
  • SCADA Basics: Memahami antarmuka pengawas mesin.
  • Cyber-Physical Systems: Integrasi IT (Information Tech) dan OT (Operational Tech).

04. Advanced OR (Stokastik)

  • Metaheuristics: Genetic Algorithms, Ant Colony Optimization, Simulated Annealing (untuk masalah optimasi yang terlalu rumit untuk matematika biasa).
  • Queueing Theory (Lanjut): M/M/1, M/M/k models untuk analisis antrean layanan atau server.
  • Markov Chains: Analisis probabilitas transisi keadaan.

05. Product Design (CAD/CAM)

  • 3D CAD Modeling: SolidWorks, AutoCAD, Fusion 360, CATIA.
  • Additive Manufacturing: Design for 3D Printing (DfAM).
  • Rapid Prototyping: Laser cutting, 3D printing operasional.
  • Product Lifecycle Management (PLM): Software manajemen siklus hidup produk (seperti Teamcenter).

06. Software Engineering Basics

  • Version Control: Git & GitHub (Manajemen kode).
  • API Integration: Cara menarik data dari sistem lain (REST API).
  • Basic Web Development: HTML/CSS (Untuk membuat dashboard kustom sederhana).

BAGIAN 3: SOFT SKILLS (INTERPERSONAL)

"Teknik Industri selalu bekerja dengan manusia. Skill ini memastikan solusi teknis Anda diterima."

01. Komunikasi & Persuasi

  • Translasi Teknis: Menjelaskan hasil regresi statistik kepada operator pabrik atau CEO dengan bahasa awam.
  • Public Speaking: Presentasi proposal proyek.
  • Technical Writing: Membuat SOP (Standard Operating Procedure) yang tidak ambigu dan mudah dibaca.
  • Storytelling with Data: Tidak sekadar menunjukkan grafik, tapi menceritakan "mengapa" grafik itu naik/turun.

02. Kepemimpinan & Manajemen

  • Conflict Resolution: Menengahi perselisihan antara departemen produksi dan QC.
  • Delegation: Mengetahui kapan harus mengerjakan sendiri dan kapan membagi tugas.
  • Empathy: Memahami kelelahan pekerja lantai pabrik (bukan melihat mereka sebagai robot).
  • Change Management: Mengelola resistensi (penolakan) karyawan terhadap sistem baru.

03. Negosiasi & Kolaborasi

  • Vendor Negotiation: Mendapatkan harga atau term pembayaran terbaik.
  • Cross-Functional Collaboration: Bekerja luwes dengan tim IT, Finance, dan HRD.
  • Stakeholder Management: Mengelola ekspektasi atasan, klien, dan tim.

BAGIAN 4: SOFT SKILLS (KOGNITIF)

"Ini adalah 'Operating System' otak seorang Engineer yang cerdas."

01. Systems Thinking

SKILL PALING KRUSIAL
  • Holistic View: Melihat dampak perubahan di satu stasiun kerja terhadap keseluruhan pabrik (The Butterfly Effect).
  • Feedback Loops: Memahami lingkaran sebab-akibat (Causal Loop Diagrams).
  • Trade-off Analysis: Memahami bahwa "lebih cepat" mungkin berarti "lebih mahal" atau "kualitas turun", dan bisa menyeimbangkannya.

02. Critical Problem Solving

  • Root Cause Analysis: 5 Whys, Fishbone (mencari akar, bukan gejala).
  • First Principles Thinking: Membedah masalah hingga ke kebenaran paling dasar, lalu membangun solusi dari sana (ala Elon Musk).
  • Hypothesis Driven: Merumuskan dugaan sebelum mengolah data, agar tidak tersesat dalam lautan data.

03. Adaptabilitas & Learning Agility

  • Unlearning: Bersedia melupakan metode lama (misal: sampling manual) demi metode baru (sensor IoT).
  • Tech-Savviness: Tidak takut mencoba software baru.
  • Resilience: Tahan banting saat proyek gagal atau data berantakan.

04. Cognitive Ergonomics

  • Mental Workload: Memahami batas beban otak manusia dalam mengambil keputusan.
  • Human-Computer Interaction (HCI): Mampu mendesain antarmuka sistem yang user-friendly agar tidak terjadi human error.

PROFIL IDEAL: The T-Shaped Engineer

Seorang ahli Teknik Industri yang cerdas dan produktif harus memiliki profil "T":

Garis Horizontal (Generalis)

Memahami dasar-dasar dari SEMUA poin di atas (tahu apa itu SPC, tahu apa itu Python, tahu cara baca Laporan Keuangan).

Garis Vertikal (Spesialis)

Menjadi SANGAT AHLI di salah satu bidang spesifik, misalnya:

  • Spesialis Supply Chain Data Scientist (Logistik + Python + ML)
  • Spesialis Smart Manufacturing (Lean + IoT + Simulation)
  • Spesialis Human Factors (Ergonomi Fisik + Kognitif + Safety)
