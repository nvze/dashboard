Visualisasi: Area Persegi dalam Setengah Lingkaran

Pilih langkah di bawah ini untuk melihat ilustrasi dari setiap perhitungan.

Pengaturan Awal

Mari kita letakkan titik tengah dasar setengah lingkaran di titik koordinat asal O(0,0). Misalkan panjang sisi persegi adalah s.

Karena persegi berada persis di tengah, sudut kiri bawah persegi (titik B) berada pada jarak s/2 ke kiri dari pusat. Tingginya adalah s (titik C).

*Klik tombol "Segitiga 1" atau "Segitiga 2" di sebelah kiri untuk melihat langkah perhitungan secara visual.

Langkah 1: Menentukan Jari-jari (R)

Perhatikan segitiga siku-siku biru △OBC. Sisi miringnya adalah jari-jari lingkaran (R). Titik C menyentuh busur lingkaran.

Alas (OB) = s/2

Tinggi (BC) = s

Teorema Pythagoras:

R² = (s/2)² + s²

R² = s²/4 + 4s²/4

R² = 5s²/4

R = (s√5) / 2

Langkah 2: Memecahkan Persamaan

Sekarang perhatikan segitiga siku-siku merah △ABC. Garis merah miring panjangnya 5 cm. Titik A adalah ujung kiri lingkaran.

Jarak O ke A = R

Jarak O ke B = s/2

Alas Segitiga (AB):

AB = Jarak(A,B) = R - s/2

Tinggi (BC) = s

Pythagoras pada △ABC:

5² = AB² + BC²

25 = (R - s/2)² + s²

Substitusi nilai R dari Langkah 1:

25 = ((s√5)/2 - s/2)² + s²

25 = s² * ((√5 - 1)/2)² + s²

...

Luas (s²) = (25 + 5√5) / 2 cm²

Luas ≈ 18.09 cm²
