Economic Order Quantity + Reorder Point

Tentukan kuantitas order optimal dan titik pemesanan ulang

EOQ + ROP
Parameter Distributor
PARAMETERSIMBOLNILAI
Permintaan tahunanD12.000 unit
Biaya pesan per orderSRp 500.000
Biaya simpan per unit/tahunHRp 200
Lead timeL7 hari
Safety stockSS100 unit
Hari kerja/tahun—300 hari
▶
LIHAT JAWABAN
1. EOQ Formula
EOQ = √(2DS / H) = √(2 × 12.000 × 500.000 / 200) = √60.000.000 = 7.746 unit
2. Frekuensi Pemesanan
N = D / EOQ = 12.000 / 7.746 ≈ ±2 kali/tahun
3. Total Inventory Cost
TC = (D/Q × S) + (Q/2 × H) = (12.000/7.746 × 500.000) + (7.746/2 × 200) = 774.600 + 774.600 = Rp 1.549.200 / tahun
4. Reorder Point (ROP)
d = D / hari = 12.000 / 300 = 40 unit/hari ROP = (d × L) + SS = (40 × 7) + 100 = 280 + 100 = 380 unit
