# Real-Time Macro & Gold Dashboard

Dashboard interaktif untuk memantau harga emas, USD/IDR, DXY, IHSG, dan Yield SBN 10 Tahun secara real-time menggunakan Python (Flask) dan yfinance.

## Cara Menjalankan di Lokal:

1. Pastikan Python sudah terinstal.
2. Buka terminal/command prompt di dalam folder ini.
3. Instal semua kebutuhan library:
   `pip install -r requirements.txt`
4. Jalankan aplikasi:
   `python app.py`
5. Buka browser dan akses `http://127.0.0.1:5000`
